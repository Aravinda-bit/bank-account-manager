# Savings Bank Simulator 🏦

A robust, console-based banking application built in Python to simulate real-world core banking operations.

## Features
* **Automated Account Creation:** Generates unique IDs using timestamp configurations.
* **Transaction Safety Filters:** Restricts withdrawals below a strict ₹1000.00 threshold.
* **Financial Forecasting:** Includes an integrated Fixed Deposit maturity calculator.
* **Crash Prevention:** Uses structured `try-except` blocks to handle text input anomalies safely.

## Technical Details
* **Language:** Python 3.x
* **Core Concepts Used:** Exception handling, Dictionary-driven mock databases, Control flow optimization.
